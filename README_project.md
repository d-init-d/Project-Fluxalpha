# Flux Alpha - Minimalist Manga/Novel Reader

## Project Overview
Flux Alpha is a modern, minimalist manga and novel reader application built with Flutter. It supports Android and Windows platforms with a focus on clean design and efficient resource management.

## Tech Stack
- **Framework**: Flutter
- **State Management**: Provider
- **Local Storage**: shared_preferences
- **Image Handling**: cached_network_image with lazy loading
- **Platforms**: Android, Windows

## Design Philosophy
- **Style**: Modern Clean & Minimalist
- **Typography**: Serif for novels, Sans-serif for UI
- **Color Scheme**: Monochrome (Black/White) dominant

## Key Features
- Manga and novel reading support
- Efficient image caching and lazy loading
- Windows keyboard shortcuts (Arrow keys)
- Clean, minimalist interface

## Project Structure
- `design_refs/` - Design references and mockups
- `prompt_library/` - Development prompts and documentation
- `test_data/` - Test data (excluded from git)
- `flux_alpha/` - Flutter source code

## Getting Started
```
cd flux_alpha
flutter run
```
